Nuevas Noticias

— App de escritorio y GitHub (22/Julio/2026) —

Lanzamos la versión 1.0.1 de la app de escritorio con varias mejoras internas y de infraestructura:

  • App de escritorio: Disponible para Linux (AppImage) y Windows (Portable) desde Descargas.
  • Auto-actualización: La app detecta automáticamente nuevas versiones al iniciar y puede actualizarse sola desde GitHub.
  • Repositorio abierto: El código está publicado en GitHub bajo licencia MIT.
  • Capítulos leídos: Los capítulos visitados ahora se marcan con un check dorado en el menú del libro.
  • Descargas inteligentes: La página de descargas detecta tu sistema operativo y te sugiere el instalador correcto.
  • Ventana persistente: La app recuerda el tamaño y posición de la ventana entre sesiones.

— Cambio de Lectura y diseño adaptativo (12/Abril/2026) —

Hoy implementamos una importante actualización visual y funcional en GreatBook Library. Hemos reconstruido la experiencia móvil y de navegación para mejorar la usabilidad:

  • Navegación Circular: Nuevo menú móvil minimalista con expansión animada "Circle Expansion" para una mejor experiencia táctil.
  • Auto-continuar Lectura: El sitio ahora recuerda automáticamente el último capítulo que visitaste de cada libro. ¡Busca el nuevo botón "CONTINUAR..." en la portada de tus obras!
  • Estado de Obras: Implementación de etiquetas dinámicas para identificar rápidamente libros en Progreso o Pausados.
  • Adaptabilidad: Interfaz unificada que se ajusta a cualquier pantalla, eliminando distracciones para centrarse en la lectura.

Nota: Las mejoras de autoguardado se expandirán a sincronización entre dispositivos en futuras versiones.
